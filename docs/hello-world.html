Hello, World!

10 Different ways to say it​

  1. Funtion:
print("Hello, world!")
  1. Variable:
msg = "Hello, world!"
print(msg)
  1. String Concatenation
greeting = "Hello, "
subject = "World!"
print(greeting + subject)
  1. Using an f-string (formatted string literal):
greeting = "Hello, "
subject = "World!"
print(f"{greeting}{subject}")
  1. Multiple print statements:
print("Hello,", end=" ")
print("World!")
  1. The sys.stdout.write method:
import sys
sys.stdout.write("Hello, World!\n")
  1. The sys.out.write with format:
import sys
sys.stdout.write("{}\n".format("Hello, World!"))
  1. Using join method:
message_parts = ["Hello," , "World!"]
print(''.join(message_parts))
  1. Using str.format method:
greeting = "Hello, {}!"
subject = "World"
print(greeting.format(subject))
  1. The repr function:
print(repr("Hello, World!"))
