# Win Tools & Utils

## Account

<table>

<tr>
  <th>Command</th>
  <th>Description</th>
</tr>

<tr>
<td>

```sh
 NET USER administrator /active:yes
```
```sh
 NET USER administrator /active:no
```
</td>
<td>

Enable/Disable Admin

</td>
</tr>

<tr>
<td>

```sh
NET USER /add LocalAdmin "Secr3t"
```

</td>
<td>

Add new user with password

</td>
</tr>

<tr>
<td>

```sh
NET LOCALGROUP administrator Ella /add
```

</td>
<td>

Add User to Local Administrator Group

</td>
</tr>
<tr>
<td>

```sh
NET USER LocalAdmin "Secr3t"
```

</td>
<td>

Change User Password

</td>
</tr>

<tr>
<td>

```sh
NET USER Ella | findstr "Password"
```

</td>
<td>

View password properties

</td>
</tr>

</table>

:::tip[Note]

These **URI** (Uniform Resource Identifier) examples only work for **Windows** and can be use in several places, such as:

- **Run Dialog**: Press Windows + R, type `ms-settings:windowsupdate`, and hit Enter. This will open the Windows Update section in the Settings app.
- **Command Prompt**: You can also run this URI from the Command Prompt using the start command:

```sh
start ms-settings:windowsupdate
```
