## Enabling math equations

To enable this feature, you can follow the detailed instruction outlined in [this article](https://docusaurus.io/docs/markdown-features/math-equations#configuration).

```bash npm2yarn
npm install --save remark-math@6 rehype-katex@7
```

## Rendering

Write inline math equations by wrapping LaTeX equations between `$`:

```tsx
$ a^2 + b^2 = c^2 $
$ E = mc^2 $
$ F = ma $
```
