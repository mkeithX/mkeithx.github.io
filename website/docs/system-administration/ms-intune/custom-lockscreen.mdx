This guide explains how to centrally manage lock screen and desktop wallpaper on Windows devices using Microsoft Intune. This helps reinforce corporate branding and communicate important information consistently.


:::tip[Note]

The sample configuration in this article focuses on **Windows Corporate Device** only and will restrict users from changing lock screen image and desktop wallpaper.

:::

## Prerequisites

- Active Microsoft Intune subscription  
- Administrative access to the Microsoft Intune admin center  
- Two wallpaper images (lock screen and desktop), stored in a publicly accessible location

<details>
<summary>Image Hosting Options</summary>

Before proceeding, host your images in a publicly accessible location. Suitable options include:

- [Azure Storage](https://portal.azure.com/)
- [Cloudflare](https://www.cloudflare.com/)
- [Cloudinary](https://cloudinary.com/)
- [Amazon S3](https://aws.amazon.com/s3/)
- [Google Cloud Storage](https://cloud.google.com/storage)

</details>

## Create a Device Configuration Profile

1. Sign in to [ Microsoft Intune Admin Center](https://aka.ms/in) 
1. Go to **Devices** > **Configuration profiles** > **Create profile**
2. **Platform:** Select **Windows 10 and later**
3. **Profile Type:** Choose **Templates** > **Device restrictions**
4. Enter a name (e.g., `Corporate Wallpaper Settings`) and an optional description

   ![Profile Creation](/img/guides/intune-lockscreen-profile-start.png)




## Configure Lock Screen and Desktop Wallpaper

### `Locked Screen Experience`

1. Under **Settings**, select **Locked Screen Experience**
2. **Lock screen image URL**: Enter the URL of the lock screen wallpaper
3. **Desktop background URL**: Enter the URL of the desktop wallpaper
4. Click **Next**

   ![Lock Screen Settings](/img/guides/intune-lockscreen-device-restriction.png)

### `Personalization`

1. Under **Settings**, select **Personalization**
2. Enter the same URLs again:
   * **Lock screen image URL**
   * **Desktop background URL**
3. Click **Next**

   ![Desktop Wallpaper Settings](/img/guides/intune-wallpaper-device-restriction.png)

## Assign the Profile to the Dynamic Group

1. In the **Assignments** step, click **Included groups** > **Add groups**
2. Select the dynamic group created earlier (e.g., `Corporate Devices`)
3. Click **Next**, then **Create**

## Verify Deployment

Once the profile is assigned, the configuration will apply to all devices in the group. Confirm by inspecting lock screen and desktop wallpaper settings on a managed device.

   ![PC Settings - Wallpaper](/img/guides/pc-settings-wallpaper.png)
   ![PC Settings - Lock Screen](/img/guides/pc-settings-lockscreen.png)
