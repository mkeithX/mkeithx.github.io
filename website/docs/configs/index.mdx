# Configuration {#website-configuration}

The SpaceHub project leverages [Docusaurus](https://docusaurus.io/) along with a few custom plugins. This section covers the site's configuration along with some basic guides about:

Install [Node.js 18+](https://nodejs.org/en/download/) and your preferred package manager ([npm](https://www.npmjs.com/), [pnpm](https://pnpm.io/), [yarn](https://yarnpkg.com/))


:::tip[workspaces]

If you are using [**yarn workspaces**](https://yarnpkg.com/features/workspaces), add `website` to your workspace list and run `yarn install` again.

:::



### Installation

```bash
npx create-docusaurus@latest website classic --typescript
```
```bash
cd website
```
```bash npm2yarn
npm run start
```
